# Frontend



## Technologies

* BootStrap 5.2.2
* React 18.2.0
* Axios 1.1.3

## Setup and Launch

* Clone the repository into a folder for frontend
* Run ```npm install```
* Run  ```npm start```
    * Runs the app in the development mode.\
      Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
